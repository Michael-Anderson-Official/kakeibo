# 家計簿

シンプルな家計簿アプリ（試作版）。1枚のHTMLで動作し、データは端末のブラウザ内（localStorage）に保存されます。

## 機能

- 収支の記録（金額・日付・カテゴリ・メモ）
- 月ごとの集計とグラフ（カテゴリ別支出、6ヶ月の収支推移）
- 定期的な収支の自動記録（家賃・給料など毎月決まった項目）
- レシート読取（対応端末ではブラウザの文字認識で金額候補を抽出）

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルドや依存関係のインストールは不要です。

## データについて

すべてのデータは端末のブラウザ内にのみ保存され、外部には送信されません。
